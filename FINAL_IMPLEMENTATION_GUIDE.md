## 技术架构

### 前端技术
- HTML5/CSS3/JavaScript
- Bootstrap 5 响应式框架
- AOS滚动动画库
- Font Awesome 图标
- 自定义CSS样式

### 后端技术
- Supabase 数据库和认证
- Flask 管理系统（本地开发）
- SQLite 本地数据库
- Vercel 部署和托管

### 数据库表结构
- products: LED产品信息
- inquiries: 客户询盘数据
- news: 新闻文章内容
- users: 管理员用户
